## 需要
  subdomain

## 流程
  1. 新增 github repo (任一名稱即可) 
  2. 在當前 jeklly 資料夾建立 ```CNAME``` 檔案，填入要轉入的 url 像是 ``` blog.hi-josh.com ```
  3. 建立 ```gh-pages``` git branch
  4. push !
